Механизм добавления геометрии:
1) Геймобджект GeometryDaat класс GeometryBoundary поставить флаг IsAddGeometry true

2)Запускаем рантайм, на маленьком поле(taskField) рисуэм фигуру. Геймобжект line покажет сколько точек в нарисованой фигуре(важно если это квадрат чтобы было 4 точки, треугольник 3 и т.д).Рисовать лучше почасовой.

-если фигура неустраивает RMB удаляет

- ">>" "<<" для просмотра уже сущесвующих фигур

- "Clear" просто очищает taskField

-"Remove" удаляет фигуру(работает с лагом по времени)

3)Нажимаем Add чтобы сохранить геометрию (одна фигура за один запуск)

4)Выходим с рантайма
